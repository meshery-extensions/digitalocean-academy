Tour the AI-Native Cloud and its five layers, learn how the Gradient AI Platform, GPU Droplets, and the Inference Engine fit together, and set up the tooling you will use throughout the academy.

Make your first model calls without managing infrastructure. Explore the model catalog, create model access keys, and use the OpenAI/Anthropic-compatible endpoint from cURL, Python, and TypeScript.

Your First AI Agent

Build, test, and publish your first managed agent on the Gradient AI Platform, then embed it in a simple web application.

Prompt Engineering Essentials

Learn the prompting techniques that make AI features predictable: instruction design, output control, few-shot patterns, and lightweight prompt evaluation.
